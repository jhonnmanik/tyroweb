Materi HTML Dasar

HTML adalah bahasa markup standar yang digunakan untuk membuat struktur dasar halaman web. Di sini Anda akan mempelajari 30 pembahasan terstruktur dari konsep dasar internet hingga pembuatan tag semantik HTML5.

1. Pengenalan Internet dan Cara Kerja Website

Website dapat tampil karena adanya proses komunikasi terarah antara Client (Browser) dan Server. Browser adalah aplikasi penjelajah yang digunakan pengguna untuk mengirimkan permintaan data, sedangkan Server adalah komputer hosting yang menyimpan seluruh file website dan mengirimkannya kembali ke Browser. 

<!-- Cara kerja website -->
<section>
  <h2>Alur Website</h2>
  <p>Browser meminta data, server mengirimkan file web.</p>
</section>

▶ Output:

Alur Website

Browser meminta data, server mengirimkan file web.

2. Sejarah Singkat dan Pengertian HTML

HTML (HyperText Markup Language) diciptakan oleh Tim Berners-Lee pada tahun 1991. HTML bukan merupakan bahasa pemrograman, melainkan bahasa markup yang menggunakan serangkaian kode tag khusus untuk menyusun dokumen teks menjadi format web terstruktur. 

<!-- Tag pembungkus konten dasar -->
<span>Dokumen HTML pertama dibuat tahun 1991.</span>

▶ Output:

Dokumen HTML pertama dibuat tahun 1991.

3. Struktur Dasar Dokumen HTML

Setiap halaman HTML wajib memiliki kerangka dasar yang terdiri dari deklarasi tipe dokumen doctype, tag html utama, tag head untuk menyimpan metadata, serta tag body untuk menampung seluruh konten visual. 

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <title>Struktur Dasar</title>
</head>
<body>
  <p>Konten utama halaman web.</p>
</body>
</html>

▶ Output:

Konten utama halaman web.

4. Atribut Dasar Elemen HTML

Atribut memberikan informasi tambahan pada elemen HTML. Penulisan atribut selalu ditempatkan di dalam tag pembuka, menggunakan format nama atribut diikuti tanda sama dengan dan nilai yang dibungkus tanda petik. 

<p id="paragraf-satu" class="teks-biru">Elemen ber-atribut id dan class.</p>

▶ Output:

Elemen ber-atribut id dan class.

5. Tag Judul Utama dan Subjudul

Untuk membuat judul tulisan, HTML menyediakan tag Heading h1 sampai h6. Tag h1 digunakan sebagai judul utama, sedangkan h2 hingga h6 bertindak sebagai subjudul pendukung dengan ukuran fisik yang mengecil berurutan. 

<h1>Judul Utama</h1>
<h2>Subjudul Pertama</h2>
<h3>Subjudul Kedua</h3>

▶ Output:

Judul Utama

Subjudul Pertama

Subjudul Kedua

6. Tag Paragraf dan Baris Baru

Teks artikel biasa ditulis di dalam tag p (paragraf). Untuk membuat perpindahan baris baru di dalam paragraf tanpa menyisakan spasi kosong yang lebar, kita menyematkan tag tunggal br. 

<p>Baris pertama teks paragraf.<br>Baris kedua langsung di bawahnya.</p>

▶ Output:

Baris pertama teks paragraf.
Baris kedua langsung di bawahnya.

7. Pemformatan Teks Tebal dan Miring

Untuk menebalkan teks yang memiliki peranan penting di dalam paragraf, kita menggunakan tag strong atau tag b. Sedangkan untuk memberikan efek miring pada teks penekanan suara, kita menggunakan tag em atau tag i. 

<p>Teks ini mengandung kata <strong>tebal penting</strong> dan teks <em>miring penekanan</em>.</p>

▶ Output:

Teks ini mengandung kata tebal penting dan teks miring penekanan.

8. Teks Sorot dan Penandaan Penting

HTML menyediakan tag mark untuk menyinari kata penting dengan warna stabilo. Selain itu, terdapat tag u untuk garis bawah, del untuk coretan teks terhapus, serta ins untuk teks sisipan baru. 

<p>Materi ini <mark>sangat penting</mark>, revisi <del>lama</del> diganti <ins>baru</ins>.</p>

▶ Output:

Materi ini sangat penting, revisi lama diganti baru.

9. Penulisan Karakter Khusus (HTML Entities)

Beberapa karakter khusus seperti tanda kurung siku siku telah dipakai oleh tag HTML. Agar browser dapat menampilkan simbol tersebut sebagai teks biasa tanpa error, kita wajib menggunakan kode HTML Entities seperti lt untuk kurang dari dan gt untuk lebih besar dari. 

<p>Simbol kurang dari ditulis sebagai &lt; dan simbol ampersand ditulis &amp;.</p>

▶ Output:

Simbol kurang dari ditulis sebagai < dan simbol ampersand ditulis &.

10. Membuat Tautan Dasar (Hyperlink)

Menghubungkan satu berkas halaman web dengan halaman web lainnya dilakukan menggunakan tag anchor (a). Atribut href wajib disematkan untuk menampung alamat URL tujuan navigasi. 

<a href="index.html">Kembali ke Beranda</a>

▶ Output:

Kembali ke Beranda

11. Membuka Tautan di Tab Baru

Saat menautkan website ke situs luar pihak ketiga, ada baiknya halaman tersebut dibuka pada tab browser baru agar pengguna tidak meninggalkan website Anda. Hal ini diatur menggunakan atribut target bernilai _blank. 

<a href="https://www.google.com" target="_blank">Cari di Google</a>

▶ Output:

Cari di Google (Membuka Tab Baru)

12. Tautan Jangkar Navigasi Internal (Anchor Link)

Anchor link digunakan untuk mengantarkan pengguna melompat secara instan ke area bagian materi tertentu di dalam satu halaman yang sama. Caranya dengan mengisi nilai atribut href menggunakan tanda pagar diikuti nama ID elemen tujuan. 

<a href="#tujuan-lompat">Lompat ke Konten Bawah</a>

▶ Output:

Lompat ke Konten Bawah

13. Menampilkan Gambar Lokal dan Eksternal

Menampilkan elemen gambar ke dalam tulisan menggunakan tag tunggal img. Kita wajib menyematkan atribut src berisi alamat file gambar, serta atribut alt berisi deskripsi teks penjelas jika gambar gagal dimuat. 

<img src="assets/gambar.jpg" alt="Foto Belajar Pemrograman">

▶ Output:

[ Foto Belajar ]

14. Mengatur Ukuran Fisik Gambar

Besar kecilnya gambar dapat diatur secara langsung menggunakan atribut width (lebar) dan height (tinggi) langsung di dalam tag img menggunakan satuan piksel atau persen. 

<img src="assets/gambar.jpg" alt="Foto Ukuran Kecil" width="100" height="60">

▶ Output:

100 x 60

15. Menyisipkan Pemutar Audio

Untuk memutar berkas lagu atau rekaman suara di halaman web, kita menyematkan tag audio yang dilengkapi dengan tag source di dalamnya. Atribut controls wajib disertakan agar tombol navigasi pemutar audio tampil di browser. 

<audio controls>
  <source src="assets/audio.mp3" type="audio/mpeg">
</audio>

▶ Output:

Pemutar Audio (controls aktif)

16. Menyisipkan Pemutar Video

Menampilkan tayangan video di web dilakukan lewat tag video. Serupa dengan audio, tag ini memerlukan sub-tag source berisi alamat berkas video, properti controls, serta properti width untuk membatasi ukuran lebarnya. 

<video width="300" controls>
  <source src="assets/video.mp4" type="video/mp4">
</video>

▶ Output:

Pemutar Video 200px

17. Membuat Daftar Tidak Berurutan (Unordered List)

Untuk menyajikan susunan informasi dalam bentuk daftar berpoin bulat biasa (tidak mementingkan urutan langkah), kita menggunakan tag pembuka ul yang membungkus daftar item di dalam tag li. 

<ul>
  <li>HTML Dasar</li>
  <li>CSS Layout</li>
</ul>

▶ Output:

  • HTML Dasar
  • CSS Layout

18. Membuat Daftar Berurutan (Ordered List)

Daftar terurut yang memiliki langkah berurutan (misalnya urutan bernomor) dibuat menggunakan tag pembuka ol yang membungkus setiap item di dalam tag li. 

<ol>
  <li>Tulis Kode HTML</li>
  <li>Buka di Browser</li>
</ol>

▶ Output:

  1. Tulis Kode HTML
  2. Buka di Browser

19. Membuat Daftar Keterangan (Description List)

Description list digunakan untuk menyajikan istilah beserta penjelasan artinya. Struktur ini dibuat menggunakan tag pembuka dl, tag dt untuk menuliskan nama istilah, serta tag dd untuk memuat isi penjelasannya. 

<dl>
  <dt>HTML</dt>
  <dd>Bahasa kerangka website.</dd>
</dl>

▶ Output:

HTML
Bahasa kerangka website.

20. Struktur Dasar Tabel

Untuk menata sebaran data secara rapi dalam baris dan kolom, kita menggunakan elemen tabel. Penulisan tabel diawali tag table, baris didefinisikan dengan tag tr, kolom judul atas dengan th, dan kolom isi data biasa dengan td. 

<table border="1">
  <tr>
    <th>Hari</th>
    <th>Kegiatan</th>
  </tr>
  <tr>
    <td>Senin</td>
    <td>Belajar HTML</td>
  </tr>
</table>

▶ Output:

Hari Kegiatan
Senin Belajar HTML

21. Menggabungkan Kolom Tabel (Colspan)

Saat merancang tabel data yang kompleks, terkadang kita butuh menggabungkan beberapa kolom horizontal menjadi satu kolom besar. Hal ini diatur menggunakan atribut colspan pada tag th atau td. 

<tr>
  <td colspan="2">Data Gabungan Kolom</td>
</tr>

▶ Output:

Kolom Gabung
A B

22. Menggabungkan Baris Tabel (Rowspan)

Selain kolom, kita juga dapat menggabungkan beberapa baris vertikal menjadi satu baris tinggi menggunakan atribut rowspan langsung pada tag td atau th bersangkutan. 

<tr>
  <td rowspan="2">Data Gabungan Baris</td>
  <td>Detail 1</td>
</tr>

▶ Output:

Baris Gabung Detail 1
Detail 2

23. Membuat Formulir Input Dasar

Formulir dibuat menggunakan tag form untuk mengumpulkan masukan dari pengguna. Untuk kolom input tulisan baris tunggal, kita menggunakan tag input dengan atribut type bernilai text. 

<form>
  <label for="nama">Nama:</label>
  <input type="text" id="nama" placeholder="Masukkan nama">
</form>

▶ Output:

24. Formulir Input Email dan Sandi

Untuk isian khusus seperti surat elektronik dan kata sandi keamanan, kita dapat menggunakan jenis tipe input yang sesuai, yaitu type="email" (memvalidasi format email otomatis) dan type="password" (menyembunyikan teks isian). 

<input type="email" placeholder="nama@email.com">
<input type="password" placeholder="Kata Sandi">

▶ Output:

25. Formulir Pilihan Bulat (Radio Button)

Radio button digunakan ketika pengguna hanya diizinkan memilih satu dari beberapa pilihan opsi yang tersedia. Pilihan bulat ini dibuat menggunakan tag input type="radio" dengan syarat wajib memiliki nilai atribut name yang sama agar terkelompok. 

<input type="radio" name="gender" value="L"> Laki-laki
<input type="radio" name="gender" value="P"> Perempuan

▶ Output:

26. Formulir Kotak Pilihan Centang (Checkbox)

Berbeda dengan radio button, kotak pilihan centang checkbox membolehkan pengguna untuk memilih lebih dari satu opsi (atau tidak memilih sama sekali) dari daftar opsi yang disediakan. Ini dibuat menggunakan input type="checkbox". 

<input type="checkbox" name="hobi" value="html"> HTML
<input type="checkbox" name="hobi" value="css"> CSS

▶ Output:

27. Membuat Dropdown Pilihan (Select)

Untuk menghemat ruang di layar formulir, daftar pilihan yang banyak sebaiknya disajikan dalam bentuk menu dropdown ke bawah. Menu pilihan ini dibuat lewat tag select yang menampung daftar elemen opsional di dalam tag option. 

<select name="kota">
  <option value="jkt">Jakarta</option>
  <option value="bdg">Bandung</option>
</select>

▶ Output:

28. Membuat Kotak Teks Paragraf Panjang

Untuk menerima masukan tulisan yang panjang (seperti kotak saran, alamat rumah lengkap, atau kolom komentar), kita menggunakan tag berpasangan textarea. Lebar dan baris kolomnya dapat disesuaikan. 

<textarea name="pesan" placeholder="Tulis masukan di sini..."></textarea>

▶ Output:

29. Membuat Tombol Kirim Form

Formulir yang telah diisi lengkap oleh pengguna membutuhkan pemicu agar dapat dikirim ke server. Hal ini dilakukan dengan membuat elemen tombol bertipe pengiriman khusus yaitu button type="submit". 

<button type="submit">Kirim Data Formulir</button>

▶ Output:

30. Tag Semantik HTML5

Tag semantik adalah tag HTML5 yang memiliki arti penamaan logis sesuai dengan peruntukan fungsinya di layout website. Penggunaan tag semantik membuat kerangka website menjadi jauh lebih bersih, terstruktur, mudah dipahami developer lain, serta sangat baik untuk optimasi mesin pencari (SEO). Contohnya tag header untuk kepala halaman, nav untuk menu navigasi, main untuk konten utama, section untuk sub-konten, dan footer untuk kaki halaman. 

<header>
  <h1>Nama Website</h1>
</header>
<main>
  <section>
    <h2>Judul Artikel</h2>
  </section>
</main>

▶ Output (Kerangka Logis Halaman):

<header> bagian atas halaman
<main> bagian konten artikel utama
<footer> bagian hak cipta kaki halaman
