HTML adalah kerangka dasar dari setiap halaman web. HTML menggunakan tag untuk mendefinisikan elemen seperti teks, gambar, dan tautan.

Membuat Tautan (Link)

Tag <a> (anchor) digunakan untuk membuat tautan yang bisa diklik untuk berpindah ke halaman lain. Atribut href digunakan untuk menentukan alamat tujuan.

Membuat Daftar (List)

Di HTML, ada dua jenis daftar utama: Unordered List (<ul>) untuk daftar menggunakan titik/bullet, dan Ordered List (<ol>) untuk daftar berurutan menggunakan angka. Isi daftarnya menggunakan tag <li>.

Membuat Tabel Data

Tabel sangat berguna untuk menampilkan data yang terstruktur. Kita menggunakan tag table sebagai pembungkus utama, tr untuk baris, th untuk judul kolom, dan td untuk isi data.

Membuat Formulir (Form)

Formulir digunakan untuk menerima input dari pengguna, seperti halaman login atau pendaftaran. Tag utamanya adalah form, yang di dalamnya terdapat berbagai jenis input dan button.


Format Teks (Text Formatting)

HTML menyediakan tag khusus untuk mengatur gaya teks tanpa perlu CSS tambahan. Beberapa yang paling umum adalah <b> untuk tebal, <i> untuk miring, dan <u> untuk garis bawah.

Pembungkus Elemen (Div dan Span)

Tag <div> dan <span> ibarat "kardus kosong" untuk merapikan elemen lain. Bedanya: div memakan satu baris penuh ke bawah (block), sedangkan span hanya membungkus teks seperlunya ke samping (inline).

Menambahkan Audio & Video

Untuk membuat web lebih hidup, kamu bisa memasukkan suara dengan <audio> atau film dengan <video>. Ingat, selalu tambahkan atribut controls agar muncul tombol play, pause, dan volume.

Elemen Semantik (Semantic HTML)

Website modern tidak hanya menggunakan tag <div> secara terus-menerus. Agar mudah dibaca oleh Google, kita menggunakan tag yang memiliki "makna" pasti, seperti <header> untuk bagian atas, <main> untuk isi utama, dan <footer> untuk bagian bawah web.
