JavaScript

JavaScript membuat website menjadi interaktif. Anda dapat menangani event klik, mengubah elemen HTML, dan melakukan perhitungan.

Interaksi DOM (Document Object Model)

Cobalah klik tombol di bawah ini untuk melihat aksi JavaScript mengubah warna elemen atau menampilkan pesan.

// Mengubah warna kotak
function ubahWarna() {
  const kotak = document.getElementById('kotak');
  kotak.style.backgroundColor = '#3B82F6';
}

// Menampilkan pesan
function tampilkanPesan() {
  tampilkanModal('Halo! Ini aksi dari JS.');
}
