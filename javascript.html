Materi JavaScript Dasar

JavaScript adalah bahasa pemrograman dinamis yang digunakan untuk membuat website menjadi interaktif, seperti mendeteksi klik mouse, merespons isian formulir, dan mengubah tulisan halaman secara langsung. Di sini Anda akan mempelajari 30 pembahasan terstruktur dari variabel, percabangan, fungsi, hingga manipulasi DOM.

1. Pengenalan JavaScript dan Perannya dalam Website

Sementara HTML bertindak sebagai struktur kerangka halaman dan CSS sebagai penghias visualnya, JavaScript (JS) berperan sebagai otak logikanya. JavaScript merupakan bahasa pemrograman dinamis yang membuat website statis pasif menjadi hidup dan interaktif merespon tindakan pengguna. 

/* Menulis logika interaktif */
console.log("JavaScript siap dijalankan!");

▶ Output (Konsol):

> JavaScript siap dijalankan!

2. Penulisan Script Internal, Eksternal, dan Tag Script

Kode JavaScript dapat dituliskan dengan dua cara utama: secara internal (ditempatkan di dalam tag script di bagian head atau sebelum penutup body), serta secara eksternal (ditulis di berkas terpisah berekstensi .js lalu dihubungkan menggunakan tag script src). 

<!-- Menghubungkan berkas JavaScript eksternal -->
<script src="scripts/script.js"></script>

▶ Output:

Berkas logika terhubung secara eksternal dan siap merespons aksi interaktif di halaman.

3. Menampilkan Output Sederhana (console.log & alert)

JavaScript menyediakan beberapa fungsi bawaan untuk menyajikan informasi. Fungsi console.log() mencetak tulisan ke jendela konsol penguji browser, sedangkan fungsi alert() memunculkan jendela dialog popup peringatan di atas layar. 

console.log("Pesan penguji konsol");
alert("Pesan peringatan popup");

▶ Output (Uji coba alert):

4. Pengenalan Variabel dan Pendeklarasian Klasik (var)

Variabel adalah kontainer memori untuk menyimpan nilai data sementara. Cara klasik mendeklarasikan variabel di JavaScript lama adalah menggunakan kata kunci var. Namun, var memiliki kelemahan dalam pembatasan ruang lingkup data (scope) sehingga rawan memicu bug. 

var nama = "Jhon";
console.log(nama);

▶ Output (Konsol):

> Jhon

5. Pendeklarasian Variabel Modern dengan Let

Kata kunci let diperkenalkan di JavaScript modern sebagai pengganti var dalam menampung data dinamis. Variabel yang dideklarasikan dengan let memiliki keunggulan pembatasan ruang lingkup block scope (hanya dikenali di dalam kurung kurawal tempatnya berada) dan nilainya dapat ditulis ulang. 

let skor = 100;
skor = 120; // Diperbolehkan mengubah nilai

▶ Output:

skor baru = 120

6. Pendeklarasian Konstanta dengan Const

Kata kunci const digunakan untuk mendeklarasikan variabel konstanta yang menampung nilai data yang bersifat mutlak dan tetap. Sekali nilai dimasukkan ke dalam variabel const, ia tidak dapat diubah kembali di baris selanjutnya (akan memicu error jika dipaksa diubah). 

const pi = 3.14;
// pi = 3.15; // Error! Konstanta tidak boleh diganti

▶ Output:

pi = 3.14

7. Tipe Data Teks (String) dan Penggabungan Teks

Tipe data String menampung tulisan huruf biasa, ditulis di antara tanda petik tunggal atau ganda. Kita dapat menggabungkan beberapa variabel string menjadi satu kalimat utuh menggunakan simbol tambah (+). 

let namaDepan = "Jhon";
let namaBelakang = "Manik";
let namaLengkap = namaDepan + " " + namaBelakang;

▶ Output:

namaLengkap = Jhon Manik

8. Tipe Data Angka (Number) dan Operasi Matematika

Tipe data Number menyimpan angka numerik bulat maupun pecahan desimal tanpa tanda petik pembungkus. Variabel angka dapat dimanipulasi dengan perhitungan aritmatika standar. 

let harga = 5000;
let jumlah = 3;
let total = harga * jumlah;

▶ Output:

total belanja = 15000

9. Tipe Data Logika (Boolean)

Tipe data Boolean hanya menampung salah satu dari dua buah kemungkinan nilai logika saja, yaitu benar (true) atau salah (false). Tipe data ini sangat krusial dalam perancangan kondisi percabangan program. 

let isLulus = true;
let isMahasiswa = false;

▶ Output:

status lulus = true | status mahasiswa = false

10. Operator Aritmatika

Operator aritmatika bertugas memanipulasi perhitungan matematis antar variabel bertipe Number: + (tambah), - (kurang), * (kali), / (bagi), serta % (Modulus/sisa hasil pembagian). 

let sisa = 10 % 3; // sisa pembagian 10 dibagi 3
console.log(sisa);

▶ Output (Konsol):

> 1

11. Operator Penugasan

Operator penugasan bertugas memasukkan atau menugaskan nilai di sebelah kanan ke dalam variabel sebelah kiri. Contohnya operator penugasan standar =, dan operator gabungan aritmatika seperti += atau -=. 

let x = 10;
x += 5; // sama artinya dengan x = x + 5

▶ Output:

x baru = 15

12. Operator Perbandingan Keras (Identitas Ketat)

Untuk memeriksa kesamaan antara dua buah nilai secara presisi, JavaScript menyediakan operator identitas ketat ===. Operator ini membandingkan kesamaan nilai fisik sekaligus kebenaran kesamaan tipe datanya. 

let cekSama = (5 === "5"); // false (karena angka tidak sama dengan teks)
console.log(cekSama);

▶ Output (Konsol):

> false

13. Operator Perbandingan Nilai

Operator perbandingan membandingkan besar kecilnya nilai angka matematika dan menghasilkan nilai Boolean: > (lebih besar), < (lebih kecil), >= (lebih besar sama dengan), dan <= (kurang dari sama dengan). 

let cekBesar = (15 >= 10);
console.log(cekBesar);

▶ Output (Konsol):

> true

14. Operator Logika Kombinasi

Operator logika bertugas menggabungkan beberapa kondisi perbandingan menjadi satu keputusan akhir: && (AND, seluruh kondisi wajib benar agar menghasilkan true), || (OR, cukup salah satu kondisi benar agar menghasilkan true), serta ! (NOT, membalikkan nilai logika). 

let lulusKuis = true;
let lunasBayar = true;
let bolehUjian = (lulusKuis && lunasBayar);

▶ Output:

bolehUjian = true

15. Struktur Percabangan Dasar (If)

Percabangan if digunakan untuk mengeksekusi suatu blok baris kode program hanya jika suatu kondisi logika yang didefinisikan di dalamnya terbukti terpenuhi (bernilai true). 

let hujan = true;
if (hujan) {
  console.log("Bawa Payung");
}

▶ Output (Konsol):

> Bawa Payung

16. Struktur Percabangan Ganda (If - Else)

Jika kondisi di dalam kriteria if tidak terpenuhi (bernilai false), maka program dapat dialihkan untuk menjalankan baris instruksi alternatif lainnya yang ditempatkan di dalam blok else. 

let nilai = 60;
if (nilai >= 75) {
  console.log("Lulus");
} else {
  console.log("Remidi");
}

▶ Output (Konsol):

> Remidi

17. Struktur Percabangan Banyak Kondisi (If - Else If - Else)

Untuk menangani pengambilan keputusan yang memiliki banyak kriteria kondisi beruntun, kita dapat menempelkan opsi else if di antara pernyataan utama if dan else. 

let nilai = 82;
if (nilai >= 85) {
  console.log("A");
} else if (nilai >= 75) {
  console.log("B"); // Kondisi terpenuhi
} else {
  console.log("C");
}

▶ Output (Konsol):

> B

18. Struktur Percabangan Pilihan Banyak (Switch Case)

Switch case digunakan untuk melakukan percabangan berdasarkan nilai pencocokan satu variabel secara spesifik. Setiap kemungkinan nilai ditandai dengan case, dan diakhiri dengan break untuk menghentikan proses pencocokan. 

let kelas = 2;
switch(kelas) {
  case 1: console.log("Pemula"); break;
  case 2: console.log("Menengah"); break; // Cocok
}

▶ Output (Konsol):

> Menengah

19. Perulangan Berdasarkan Hitungan (For)

Perulangan for dioperasikan saat kita sudah tahu pasti seberapa banyak baris kode program harus diulang. Tiga bagian utamanya adalah inisialisasi awal variabel pencacah, batas akhir pencacah, serta penambahan nilai pencacah (increment). 

for (let i = 1; i <= 3; i++) {
  console.log("Pengulangan " + i);
}

▶ Output (Konsol):

> Pengulangan 1
> Pengulangan 2
> Pengulangan 3

20. Perulangan Berdasarkan Kondisi (While)

Berbeda dengan for, perulangan while digunakan untuk mengulang sekumpulan kode selama suatu syarat kondisi di dalamnya terbukti masih bernilai benar (true) tanpa harus tahu pasti berapa putaran yang terjadi. 

let count = 1;
while (count <= 2) {
  console.log("Cetak: " + count);
  count++;
}

▶ Output (Konsol):

> Cetak: 1
> Cetak: 2

21. Pengenalan Fungsi Tanpa Parameter

Fungsi (Function) adalah sebuah blok kode program modular mandiri yang sengaja dikelompokkan bersama untuk mengeksekusi suatu perintah kerja tertentu secara berulang kali hanya dengan menyebutkan nama fungsinya. 

function sapaDunia() {
  console.log("Halo Dunia!");
}
sapaDunia(); // Memanggil fungsi

▶ Output (Konsol):

> Halo Dunia!

22. Fungsi dengan Parameter Masukan (Argumen)

Sebuah fungsi dapat dirancang agar dapat menerima nilai variabel masukan dari luar kontainernya saat dipanggil. Variabel penampung nilai masukan ini diletakkan di dalam tanda kurung dan disebut parameter. 

function haloNama(nama) {
  console.log("Halo, " + nama);
}
haloNama("Jhon");

▶ Output (Konsol):

> Halo, Jhon

23. Fungsi yang Mengembalikan Nilai (Return)

Fungsi dapat mengembalikan suatu nilai hasil pengolahan data ke bagian baris kode pemanggilnya menggunakan kata kunci return. Pernyataan return juga bertugas langsung menghentikan jalannya fungsi seketika itu juga. 

function kuadrat(angka) {
  return angka * angka;
}
let hasil = kuadrat(4);

▶ Output:

hasil = 16

24. Pengenalan Tipe Data Larik (Array)

Array adalah struktur data berurutan khusus yang dapat menyimpan sekumpulan beberapa nilai data sekaligus di dalam satu nama variabel tunggal saja secara terurut. 

let data = ["HTML", "CSS", "JS"];
console.log(data);

▶ Output (Konsol):

> ["HTML", "CSS", "JS"]

25. Mengakses dan Mengubah Elemen Array

Setiap elemen di dalam array diakses menggunakan nomor indeks pencacahnya. Indeks array di dalam JavaScript selalu diawali dari angka 0 untuk elemen pertamanya. Nilai elemen di dalam indeks tersebut dapat diubah di baris kode selanjutnya. 

let buah = ["Apel", "Mangga"];
buah[0] = "Jeruk"; // Mengubah elemen pertama menjadi Jeruk
console.log(buah[0]);

▶ Output (Konsol):

> Jeruk

26. Mengetahui Jumlah Elemen Array dan Menambah Data

Untuk mengetahui total jumlah seluruh item data di dalam variabel array, kita menggunakan properti length. Dan untuk menambahkan item data baru pada urutan paling belakang array, kita menggunakan metode push(). 

let kota = ["Jakarta"];
kota.push("Bandung");
console.log(kota.length); // 2

▶ Output (Konsol):

> 2

27. Pengantar DOM (Document Object Model)

DOM adalah struktur representasi visual dokumen web di mata JavaScript. Menggunakan DOM, berkas HTML dipetakan menjadi pohon objek di mana JavaScript dapat mengakses dan memodifikasi strukturnya secara dinamis. 

/* Struktur DOM memetakan body, div, dll */
console.log(document.body);

▶ Output:

Struktur objek dokumen siap dimanipulasi secara real-time.

28. Mengakses Elemen HTML Lewat DOM

Untuk memanipulasi elemen HTML tertentu, JavaScript harus menyeleksinya terlebih dahulu. Perintah pencarian elemen paling populer berdasarkan ID unik adalah menggunakan fungsi document.getElementById(). 

const elemen = document.getElementById("judul");

▶ Output:

Elemen ID berhasil dijangkau dan siap diberikan aksi.

29. Mengubah Konten Teks dan Gaya CSS Melalui DOM

Setelah elemen dijangkau lewat DOM, kita dapat mengganti isi konten tulisannya secara langsung dengan memodifikasi properti .innerText, serta mengubah hiasan visualnya melalui properti .style. 

const box = document.getElementById("demo");
box.innerText = "Teks Diubah!";
box.style.color = "#1e3a8a";

▶ Output (DOM Demo):

Teks asli sebelum diubah

30. Menangkap Aksi Pengguna Lewat Event Listener

Event adalah setiap aksi fisik yang dilakukan pengguna pada halaman website (seperti mengeklik tombol, mengetik kolom form). JavaScript memantau dan menangkap setiap aksi kejadian ini menggunakan metode addEventListener() untuk memicu eksekusi fungsi respons. 

const tombol = document.getElementById("btn");
tombol.addEventListener("click", function() {
  alert("Event terpicu!");
});
