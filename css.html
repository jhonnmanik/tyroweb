Materi CSS Dasar

CSS digunakan untuk mengatur dan menghias tampilan visual dari halaman website. Di sini Anda akan mempelajari 30 pembahasan terstruktur dari selector dasar, Box Model, tata letak modern, hingga media queries responsif.

1. Pengenalan CSS dan Struktur Aturan Sintaks

CSS (Cascading Style Sheets) bertugas untuk merancang gaya estetika visual halaman web. Sintaks dasar penulisan kode CSS terdiri dari Selector (elemen mana yang ingin diberi hiasan) diikuti oleh pasangan Property (sifat gaya yang ingin diubah) dan Value (nilai perubahan gayanya). 

/* Aturan sintaks dasar CSS */
h1 {
  color: #4f46e5;
  font-size: 24px;
}

▶ Output:

Judul Bergaya CSS

2. Cara Menghubungkan CSS ke HTML (Inline, Internal, & External)

Ada tiga cara menghubungkan gaya CSS ke halaman HTML: Inline CSS (ditulis langsung di atribut style elemen), Internal CSS (ditulis di dalam tag style pada head), serta External CSS (ditulis di berkas terpisah berekstensi .css lalu dihubungkan menggunakan tag link). 

<!-- Menghubungkan berkas CSS eksternal -->
<link rel="stylesheet" href="styles/style.css">

▶ Output:

Berkas gaya terhubung secara eksternal dan siap merender tampilan visual secara rapi.

3. Selector CSS Berdasarkan Tag Elemen

Selector Tag memilih seluruh elemen halaman HTML berdasarkan nama tag dasarnya. Misalnya, menuliskan selector p akan langsung mengubah gaya seluruh paragraf teks di halaman web secara bersamaan tanpa terkecuali. 

p {
  color: #374151;
}

▶ Output:

Paragraf teks ini terpengaruh secara global oleh selector tag p.

4. Selector CSS Berdasarkan Class

Selector Class memilih elemen-elemen yang memiliki atribut class tertentu. Di dalam berkas CSS, penulisan nama class wajib diawali dengan tanda titik (.). Keunggulan selector ini adalah sangat fleksibel karena dapat disematkan berulang kali pada banyak elemen. 

.teks-tebal {
  font-weight: bold;
}

▶ Output:

Teks ini menggunakan class teks-tebal sehingga memiliki ketebalan visual.

5. Selector CSS Berdasarkan ID

Selector ID menargetkan satu elemen spesifik secara unik di halaman web. Penulisan nama ID di dalam berkas CSS wajib diawali dengan tanda pagar (#). ID bersifat eksklusif dan hanya boleh disematkan pada satu elemen saja di dalam satu dokumen HTML. 

#judul-utama {
  font-size: 28px;
}

▶ Output:

Elemen Unik ber-ID #judul-utama

6. Selector Universal dan Pengelompokan Selector

Selector Universal disimbolkan tanda bintang (*) dan berguna untuk mengubah seluruh elemen halaman tanpa terkecuali. Selain itu, Anda dapat mengelompokkan beberapa selector berbeda agar memiliki gaya aturan yang sama sekaligus dengan memisahkannya menggunakan tanda koma (,). 

/* Pengelompokan selector */
h1, h2, h3 {
  color: #1e3a8a;
}

▶ Output:

Heading 1, 2, dan 3 otomatis berwarna biru navy secara terkelompok.

7. Pewarnaan Teks Menggunakan Nama dan HEX Kode

Mengubah warna teks dilakukan melalui properti color. Kita dapat menentukan warnanya dengan menyebutkan nama warna dasar bahasa Inggris (seperti blue atau red) ataupun menggunakan penomoran Hexadecimal (HEX) enam digit diawali tanda pagar. 

p {
  color: #4f46e5;
}

▶ Output:

Teks berwarna ungu indigo menggunakan HEX #4f46e5.

8. Pewarnaan Teks Menggunakan RGB dan HSL

Selain HEX, CSS menyediakan opsi penulisan format warna modern menggunakan kombinasi parameter RGB (Red, Green, Blue) ataupun HSL (Hue, Saturation, Lightness). 

p {
  color: rgb(79, 70, 229);
}

▶ Output:

Teks berwarna menggunakan format RGB.

9. Mengatur Warna Latar Belakang (Background Color)

Untuk mengubah warna alas atau latar belakang dari suatu persegi kontainer elemen HTML, kita menggunakan properti background-color. 

.box {
  background-color: #dbeafe;
}

▶ Output:

Latar Belakang Biru Lembut (#dbeafe)

10. Membuat Warna Latar Belakang Gradasi

CSS membolehkan pembuatan efek transisi warna gradasi modern yang memukau tanpa bantuan gambar menggunakan properti background dengan fungsi penulisan linear-gradient. 

.box-gradasi {
  background: linear-gradient(135deg, #4f46e5, #9333ea);
}

▶ Output:

Gradasi linear-gradient Indigo & Ungu

11. Mengatur Jenis Keluarga Huruf (Font Family)

Mengatur jenis keluarga huruf tulisan menggunakan properti font-family. Kita dapat menyusun beberapa opsi nama font cadangan sebagai antisipasi jika font utama tidak tersedia di komputer sistem operasi pengguna. 

p {
  font-family: Arial, Helvetica, sans-serif;
}

▶ Output:

Teks dirender rapi menggunakan huruf Arial sans-serif.

12. Mengatur Ukuran dan Ketebalan Huruf

Besar kecilnya teks diatur menggunakan properti font-size (dengan satuan piksel px, em, atau rem). Tingkat ketebalan huruf diatur menggunakan font-weight (seperti normal, bold, atau angka ketebalan 100 sampai 900). 

.teks-besar-tebal {
  font-size: 18px;
  font-weight: 800;
}

▶ Output:

Teks Berukuran 16px dengan Ketebalan Tebal 800

13. Mengatur Tinggi Baris Tulisan (Line Height)

Properti line-height bertugas mengatur jarak spasial vertikal antar baris tulisan di dalam paragraf agar teks tidak saling berhimpitan padat sehingga jauh lebih nyaman untuk dibaca. 

p {
  line-height: 1.8;
}

▶ Output:

Teks paragraf ini menggunakan line-height 1.8. Jarak spasial baris atas dan baris bawah sangat renggang dan nyaman dibaca mata pembaca.

14. Mengatur Perataan Teks (Text Align)

Properti text-align digunakan untuk memosisikan sebaran horisontal teks tulisan di dalam kontainer: left (rata kiri), center (rata tengah), right (rata kanan), dan justify (rata kiri kanan secara seimbang). 

p {
  text-align: center;
}

▶ Output:

Teks terposisikan tepat di tengah-tengah kontainer (center).

15. Konsep Utama Box Model CSS

Di dunia CSS, seluruh elemen HTML dianggap sebagai sebuah kotak persegi yang memiliki empat buah lapisan menyelimuti satu sama lain dari dalam ke luar: Content (isi teks/gambar), Padding (jarak dalam), Border (garis tepi), dan Margin (jarak luar). 

.box {
  padding: 15px;
  border: 2px solid #4f46e5;
  margin: 15px;
}

▶ Output:

MARGIN (Luar)
BORDER & PADDING (Dalam) -> CONTENT

16. Mengatur Jarak Bagian Dalam (Padding)

Padding adalah ruang spasial di bagian dalam antara konten teks dengan garis tepi pembatas (border) kotak elemen tersebut. Padding dapat diatur nilainya secara terpisah untuk sisi atas, kanan, bawah, dan kiri. 

.box {
  padding-left: 30px;
  padding-top: 10px;
}

▶ Output:

Kotak ini memiliki rongga padding kiri sebesar 30px.

17. Mengatur Garis Tepi (Border)

Border adalah garis pembatas tepi fisik kotak elemen. Kita dapat menentukan tipe garis (border-style seperti solid, dashed), ketebalan garis (border-width), warna garis (border-color), serta kelengkungan sudut persegi kotak tersebut menggunakan properti border-radius. 

.box-bulat {
  border: 2px solid #264de4;
  border-radius: 12px;
}

▶ Output:

Kotak bersudut melengkung (border-radius: 12px)

18. Mengatur Jarak Bagian Luar (Margin)

Margin adalah batas rongga spasial di bagian luar elemen untuk memisahkan kotak elemen tersebut dari elemen lainnya. Margin luar yang sama besar di sisi kiri dan kanan dapat diatur dengan menuliskan nilai auto guna memosisikan elemen persegi tepat di tengah-tengah secara horisontal. 

.tengah {
  margin: 0 auto;
  width: 200px;
}

▶ Output:

Kotak terpusat (margin: auto)

19. Memahami Display Block

Elemen dengan gaya display block (secara default disematkan pada tag div, p, h1) akan selalu mengambil lebar horizontal penuh yang tersedia di layar halaman web dan memaksa elemen selanjutnya agar berpindah ke baris baru di bawahnya. 

.blok {
  display: block;
  background-color: #cbd5e1;
}

▶ Output:

Elemen Block mengisi baris penuh kontainer.

20. Memahami Display Inline

Elemen inline (secara default dimiliki tag span, strong, a) hanya memakan lebar halaman secukupnya sesuai dengan ukuran konten yang dibungkusnya, serta sejajar horisontal dengan elemen sekelilingnya tanpa membuat baris baru. Elemen inline tidak dapat diatur nilai lebar width dan tingginya. 

.sejajar {
  display: inline;
  background-color: #fca5a5;
}

▶ Output:

Inline A Inline B sejajar horisontal

21. Memahami Display Inline-Block

Inline-block adalah tipe hibrida yang memadukan keunggulan kedua display sebelumnya. Elemen sejajar secara horizontal di satu baris yang sama seperti inline, namun fisiknya tetap dapat diatur nilai lebar width dan tinggi height-nya layaknya elemen block. 

.kotak-sejajar {
  display: inline-block;
  width: 100px;
  height: 40px;
}

▶ Output:

Sejajar 100px
Sejajar 100px

22. Menyembunyikan Elemen (Display None & Visibility Hidden)

Ada dua cara menyembunyikan elemen: display: none (menghilangkan elemen secara total dari tata letak halaman sehingga ruang kosongnya tidak disisakan), serta visibility: hidden (menyembunyikan visual elemen tetapi tetap menyisakan ruang kosong fisik dari ukuran elemen tersebut). 

.hilang {
  display: none;
}

▶ Output:

Terlihat [ Elemen display:none tersembunyi tanpa memakan ruang ]

23. Posisi Static dan Relative

Static adalah peletakan koordinat default alami yang mengikuti aliran pemuatan halaman. Sedangkan relative membolehkan kita menggeser posisi fisik elemen (menggunakan properti top, left, dll.) dihitung relatif dari koordinat aslinya berada. 

.relatif {
  position: relative;
  left: 20px;
}

▶ Output:

Geser ke Kanan 20px (Relative)

24. Posisi Absolute

Elemen dengan gaya position: absolute akan terbang melayang terlepas dari aliran dokumen alami. Koordinat peletakannya dihitung relatif terhadap batas sisi kontainer induk pembungkus terdekatnya yang berposisi selain static (biasanya diatur relative). 

.induk { position: relative; }
.anak {
  position: absolute;
  bottom: 5px;
  right: 5px;
}

▶ Output:

Pojok Kanan Bawah

25. Posisi Fixed dan Sticky

Fixed mengunci peletakan elemen pada koordinat layar browser sehingga melayang diam di sana dan tidak ikut bergerak saat halaman di-scroll ke bawah. Sedangkan sticky bertindak normal, lalu akan terkunci menempel di posisi atas setelah menyentuh batas koordinat scroll-nya. 

.menu-atas {
  position: sticky;
  top: 0;
}

▶ Output:

Sistem navigasi sticky menempel saat digulir ke atas.

26. Pengantar Tata Letak Flexbox

Flexbox (Flexible Box Layout) adalah sistem tata letak satu dimensi (mendatar atau menurun) yang sangat baik untuk mendistribusikan ruang spasial antar elemen secara dinamis dan responsif. Pengaktifan dilakukan dengan mendeklarasikan properti display: flex pada kontainer induknya. 

.wadah-flex {
  display: flex;
}

▶ Output:

Flex Item A
Flex Item B

27. Perataan Flexbox Horisontal (Justify Content)

Untuk menyebarkan dan memosisikan letak seluruh elemen anak di sepanjang garis horizontal sumbu utama flexbox, kita menggunakan properti justify-content (dengan nilai seperti flex-start, center, flex-end, space-between). 

.wadah {
  display: flex;
  justify-content: space-between;
}

▶ Output (Flexbox Alignment Simulator):

1
2

28. Perataan Flexbox Vertikal (Align Items)

Untuk meratakan letak vertikal elemen anak sejajar tegak lurus sumbu utama flexbox, kita menggunakan properti align-items (dengan opsi seperti flex-start, center, atau flex-end). 

.wadah {
  display: flex;
  align-items: center;
}

▶ Output:

Tengah Vertikal
Rata Tengah

29. Pengantar Tata Letak Grid Dua Dimensi

CSS Grid adalah raja tata letak dua dimensi yang membagi elemen visual dalam baris dan kolom secara bersamaan. Pengaktifan dilakukan dengan display: grid, dan pembagian kolomnya dilakukan lewat properti grid-template-columns. 

.kisi-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 8px;
}

▶ Output:

Kotak 1
Kotak 2
Kotak 3

30. Desain Web Responsif Menggunakan Media Queries

Responsive Web Design memastikan website tampil rapi saat dibuka di HP yang berlayar sempit maupun komputer yang lebar. Alat utama CSS untuk mendeteksi lebar resolusi layar pembuka ini adalah Media Query (@media). Gaya CSS orisinal akan ditimpa jika batas resolusi layar tersebut tercapai. 

@media (max-width: 768px) {
  .kisi {
    grid-template-columns: 1fr;
  }
}

▶ Output:

Mode Desktop (Mendatar)
Kolom 1
Kolom 2
Mode HP (Menurun)
Kolom 1
Kolom 2
