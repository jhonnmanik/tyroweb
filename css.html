CSS (Cascading Style Sheets)

CSS digunakan untuk mendesain dan mempercantik halaman web. Anda bisa mengatur warna, ukuran, tata letak, hingga animasi.

Membuat Tombol Menarik

Berikut adalah cara kita memberikan gaya pada elemen HTML agar terlihat lebih modern.

.btn-keren {
    background-color: #10B981;
    color: white;
    padding: 10px 20px;
    border: none;
    border-radius: 20px;
    font-size: 16px;
    box-shadow: 0 4px 6px rgba(0,0,0,0.2);
    cursor: pointer;
}
